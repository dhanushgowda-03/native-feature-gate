Evaluation Strategies

GLOBAL_ON
Flag enabled for every user, always.
parameters: null or {}
USER_WHITELIST
Enabled only if context.userId is in the explicit list.
parameters: {"userIds": ["u1", "u2", ...]}
PERCENTAGE_ROLLOUT
Deterministic hash(userId:flagKey) % 100 < percentage.
parameters: {"percentage": 10}
